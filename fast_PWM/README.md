В программе модулируется ШИМ
На 3 выводе **PORTB** изначально логическая "1". В цикле на каждой итерации модулируется скважность равная "i"
Далее при совпадении **TCNT0** c **OCR0** вывод **OC0** сбрасывается в ноль.
